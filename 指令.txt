1.  vim可以进入文件里面 进入文件里面按i键进行输入 按:wq可以经行退出
2.  ls 				查看文件
3.  ls -a 			查看所有文件
4.  cat 文件名			查看文件里面的信息
5.   git add 文件名	 	可以进行提交到缓存区
6.   git status 			查看是不是进入了缓存区
7.   git rm --cached 文件名 	删除在缓存区的文件
8.   git add . 			添加当前目录下的所有文件
9.   git commit -m 注释		提交到历史区
10. git log 			查看历史区的记录
11.  git reset --hard HEAD^	返回上一个版本或者加多个尖角号返回多个版本
12.  git reser --hard HEAD~1	返回上一个版本或者加数字返回多个版本
